Other OS Operations

The following are miscellaneous operating system operations.

system string
Executes the string in the underlying OS.
 
>(system "date")
Sun Mar 23 13:48:46 PDT 2008

nil
quit
Exits the interpreter.
 
>(quit)
--exits--
